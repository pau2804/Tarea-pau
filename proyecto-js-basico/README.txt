Este proyecto incluye varias funcionalidades implementadas en JavaScript, que se pueden ejecutar en un navegador web. A continuación, se describen las funcionalidades y cómo ejecutar el proyecto.

## Instrucciones para Ejecutar el Proyecto

1. **Clonar el Repositorio**: Si has descargado el proyecto desde un repositorio, asegúrate de clonar el repositorio en tu máquina local.

2. **Abrir el Archivo `index.html`**: Navega a la carpeta del proyecto y abre el archivo `index.html` en tu navegador web preferido. Puedes hacerlo haciendo doble clic en el archivo o arrastrándolo a la ventana del navegador.

3. **Interacción con las Funcionalidades**:
   - **Calculadora Básica**: Utiliza los formularios para realizar operaciones matemáticas básicas como suma, resta, multiplicación y división.
   - **Contador de Palabras**: Ingresa una frase en el área de texto y presiona el botón correspondiente para contar el número de palabras.
   - **Filtrado de Números Pares**: Proporciona un array de números y utiliza la función para filtrar y mostrar solo los números pares.
   - **Calculadora de Promedio**: Ingresa un array de calificaciones y obtén el promedio junto con un mensaje de aprobado o reprobado.
   - **Juego de Adivinanza**: Juega ingresando un número y recibe pistas sobre si es mayor, menor o correcto.
   - **Manipulación de Objetos**: Interactúa con el objeto `estudiante` para calcular y mostrar el promedio de sus calificaciones.
   - **Tabla de Multiplicar**: Ingresa un número y genera su tabla de multiplicar.

## Funcionalidades Implementadas

- **Calculadora Básica**: Sumar, restar, multiplicar y dividir.
- **Contador de Palabras**: Cuenta las palabras en una frase.
- **Filtrado de Números Pares**: Filtra un array para mostrar solo números pares.
- **Calculadora de Promedio**: Calcula el promedio de un array de calificaciones.
- **Juego de Adivinanza**: Lógica para adivinar un número.
- **Manipulación de Objetos**: Cálculo del promedio de un objeto `estudiante`.
- **Tabla de Multiplicar**: Genera la tabla de multiplicar de un número.

Asegúrate de tener un navegador moderno para una mejor experiencia. ¡Diviértete explorando las funcionalidades de este proyecto!
